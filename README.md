# เกมส์ทายตัวเลข (Number Guessing Game)

## ความเป็นมา

โปรแกรมเกมส์ทายตัวเลข 1-1000 เขียนด้วย HTML, CSS และ Vanilla JavaScript

## Difficulty Level Challenge ระดับความยาก

- 1 - 10
- 1 - 50
- 1 - 100
- 1 - 1000

## เวอร์ชัน

- v1: เวอร์ชันพื้นฐาน
- v2: เพิ่ม Bootstrap, validation, counter
- v3: เพิ่มระดับความยากในการเล่น

## วิธีการใช้

1. เปิด `index.html` ในเบราว์เซอร์
2. เลือกระดับความยากของการทายเลข
3. พิมพ์ตัวเลข แล้วคลิก "ทาย"
4. โปรแกรมจะบอกว่าสูงไป/ตํ่าไป/ถูกต้อง

## ข้อกําหนด

- HTML5
- CSS3 (Bootstrap 5)
- JavaScript ES6+
